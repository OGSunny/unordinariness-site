Quick Start Guide

  1. Invite the Bot: Click the "Add to Discord" button on our homepage. Choose the server you want to protect and grant the required permissions. The bot needs administrator permissions to function correctly.
  2. Run the Setup Command: Once the bot is in your server, navigate to a text channel and type `/setup moderate`. This command will automatically create a moderation log channel, a private owner log channel, and set up basic security configurations.
  3. Explore Features: Your server is now protected! You can start using commands like `/warn`, `/mute`, and `/kick`. To see a full list of commands, run `/help` or visit the Command Reference section below.
  4. Customize Your Settings: Use the `/config` command to fine-tune settings like anti-raid sensitivity, content filter levels, and punishment thresholds.

Command Reference

Vertex Security supports both slash (/) and prefix (!) commands for convenience.

Moderation Commands

/warn [user] [reason]

Issues a formal warning to a user.

/mute [user] [duration] [reason]

Silences a user for a specified period.

/kick [user] [reason]

Removes a user from the server.

/ban [user] [reason]

Permanently bans a user from the server.

/lock [channel]

Disables sending messages in a channel.

/unban [user ID]

Unbans a user from the server.

Security & Setup Commands

/setup [level]

Initializes the bot with a security level (relaxed, moderate, strict).

/config [setting] [value]

Customizes various bot settings.

/security-level [level]

Changes the server's overall security level.

Advanced Configuration

Vertex Security offers granular control to match your server's unique needs. Use the `/config` command to fine-tune these settings:

Anti-Raid Settings

Control the sensitivity of the anti-raid system. Adjust the number of members joining per minute required to trigger lockdown. You can also specify a custom message for the CAPTCHA and a list of roles to bypass the verification.

Content Filter

Enable or disable the filter, and set punishment levels for repeat offenders. Choose to issue warnings, mutes, or an immediate ban. You can also add custom words or phrases to a blacklist.

Logging and Permissions

Designate separate channels for general moderation logs and private owner logs. Customize which roles have permission to use moderation commands and view sensitive log channels.
