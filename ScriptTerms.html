⚠️ Important Notice

Use all scripts at your own risk. We are not responsible for any account actions or issues.

1. Script Usage Guidelines

Before Using Any Script

  • Always test scripts in private servers when possible
  • Read and understand what the script does
  • Be aware of the risks involved with script execution
  • Keep backups of important game data

Potential Risks

Using scripts may result in temporary or permanent account restrictions, loss of game progress, or other consequences determined by game developers.

2. Script Sharing Rules

What's Allowed

  • Sharing legitimate, safe scripts
  • Educational content about scripting
  • Open source scripts with proper attribution
  • Scripts you have created yourself

Strictly Prohibited

  • Malicious code or backdoors
  • Scripts designed to harm users or steal data
  • Copyrighted scripts without permission
  • Scripts that violate platform terms of service

Credit Where Due

Always credit the original creators when sharing scripts. If you modify someone's work, acknowledge both the original author and your modifications.

3. Community Responsibility

Our community operates on trust and mutual respect. When sharing or using scripts:

  • Report any suspicious or malicious content immediately
  • Help newer users understand the risks involved
  • Share your experiences and knowledge responsibly
  • Respect others' intellectual property

Zero Tolerance Policy

Sharing malicious scripts will result in immediate and permanent removal from our community. We take the safety of our members seriously.

4. Disclaimer

Vertex Hub and its staff are not responsible for:

  • Any consequences resulting from script usage
  • Account actions taken by game platforms
  • Loss of game progress or virtual items
  • Technical issues caused by script execution
  • Content shared by community members

Scripts are provided "as-is" without any warranty. Users assume all responsibility and risk when choosing to execute any code.

5. Best Practices

For Script Users

  • Start with simple, well-known scripts
  • Learn to read and understand code basics
  • Use reputable script executors
  • Stay informed about platform policies

For Script Developers

  • Write clean, commented code
  • Test thoroughly before sharing
  • Provide clear documentation
  • Be responsive to user feedback

Learning Resources

Take advantage of our community's knowledge. Ask questions, participate in discussions, and help others learn. Together we can create a safer, more informed community.
