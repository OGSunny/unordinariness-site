Quick Setup Guide

Add RoxTether to Your Server

Invite the bot with required permissions (Manage Roles, Send Messages, Use Slash Commands) to start verification services.

Configure Group Bindings

Use /bind [group_id] [min_rank] [discord_role] to link Roblox groups with Discord roles automatically.

Members Verify Accounts

Users run /verify and complete secure Roblox OAuth authentication to link their accounts.

Enjoy Automated Roles

Roles are assigned instantly based on group membership, with automatic updates as member status changes.
